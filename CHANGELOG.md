# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

El formato se basa en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto se adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-21

### Añadido
- Creación del repositorio en GitHub.
- Documentación inicial (`README.md` y `CHANGELOG.md`).
- Implementación de `MultiStepForm` con validación Zod.
- Componentes de la landing page: `Navbar`, `Hero`, `Features`, `Testimonials`, `FAQ` y `Footer`.
- Sistema de diseño basado en Tailwind CSS 4 con tema personalizado (Light Green & Slate).
- Animaciones de transición para los pasos del formulario con Framer Motion.
- Configuración de SEO base en `index.html`.

### Cambios
- Optimización de la sección Hero con fondo de coche de alta calidad y gradiente de transparencia.
- Ajuste de colores primarios a un verde esmeralda más profesional (`#B9F8CF` y `#059669`).

### Corregido
- Problemas de hidratación menores en componentes estáticos.
